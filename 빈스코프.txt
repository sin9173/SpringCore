싱글톤
 - 기본 스코프
 - 스프링 컨테이너의 시작과 종료까지 유지

프로토타입
 - 스프링 컨테이너가 빈의 생성과 의존관계 주입까지만 관여
 - 의존관계 주입이 끝나면 더이상 관리하지 않음
 - 종료 메소드가 호출되지 않음
 - 빈을 조회할때 빈 생성

웹 스코프
request : HTTP 요청이 들어오고 응답을 줄때까지 유지되는 스코프
session : 웹세션이 생성되고 종료될 때 까지 유지되는 스코프
application : 웹 서블릿 컨텍스트와 같은 범위로 유지
websocket : 웹 소켓과 동일한 생명주기를 가지는 스코프
